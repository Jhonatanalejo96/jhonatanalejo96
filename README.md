# Hola, soy Jhonatan Piza

### 🚀 QA Automation Engineer | ISTQB® Certified | Ingeniero Mecatrónico

Soy un ingeniero apasionado por la calidad del software con más de 4 años de experiencia en los sectores financiero, telecomunicaciones y retail. Me especializo en diseñar, construir y mantener frameworks de automatización escalables y eficientes. Mi enfoque va más allá de encontrar bugs: busco optimizar los pipelines de CI/CD y garantizar entregas continuas, seguras y de alto impacto para el negocio.

## 📂 Mi Catálogo de Automatización (¿Qué encontrarás aquí?)

Este perfil funciona como mi portafolio técnico y bitácora de aprendizaje. Aquí documento mis implementaciones, guías de estudio estructuradas y soluciones arquitectónicas para pruebas de software:

- **Arquitecturas Escalables:** Proyectos base construidos desde cero aplicando buenas prácticas, Clean Code y patrones de diseño como **Page Object Model (POM)** y **Screenplay**.
- **Frameworks Modernos & BDD:** Implementaciones detalladas y escenarios orientados a negocio integrando **Playwright**, **Cypress** y **Cucumber** (Gherkin) con ejecuciones en la nube vía **GitHub Actions**.
- **Pruebas E2E y Regresión:** Automatización madura con Selenium (Java/Python) asegurando la cobertura total de flujos críticos.
- **Performance & API Testing:** Scripts de pruebas de carga, estrés y concurrencia utilizando **JMeter** y **k6** para identificar cuellos de botella antes de salir a producción.
